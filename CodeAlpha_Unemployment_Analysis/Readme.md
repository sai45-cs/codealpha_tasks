📊 Unemployment Analysis in India

A data analysis project that explores unemployment trends in India using Python, Pandas, and Matplotlib. The project analyzes regional unemployment rates, identifies trends over time, and examines the impact of the COVID-19 pandemic on unemployment.

⸻

📌 Project Overview

The objective of this project is to analyze unemployment data in India and extract meaningful insights through data preprocessing, statistical analysis, and visualization.

The project focuses on:

* Exploring unemployment trends over time
* Comparing unemployment rates across regions
* Analyzing the impact of COVID-19 on unemployment
* Visualizing key patterns using graphs

⸻

📊 Dataset

The project uses the Unemployment in India dataset, which contains unemployment statistics across various regions and time periods.

Features

Feature	Description
Region	State/Region name
Date	Date of observation
Estimated Unemployment Rate (%)	Percentage of unemployed individuals
Estimated Employed	Number of employed individuals
Estimated Labour Participation Rate (%)	Labour force participation rate
Area	Rural or Urban classification

🔍 Analysis Performed

The project performs the following analyses:

1. Load and clean the dataset.
2. Convert date columns into datetime format.
3. Check dataset dimensions and missing values.
4. Generate summary statistics.
5. Calculate average unemployment rates by region.
6. Visualize unemployment trends over time.
7. Analyze unemployment during the COVID-19 period.
8. Identify regions with the highest unemployment rates.

⸻

📈 Visualizations

The project generates the following visualizations:

* Average Unemployment Rate Over Time
* Unemployment Rate During COVID-19
* Top 10 Regions by Average Unemployment Rate
